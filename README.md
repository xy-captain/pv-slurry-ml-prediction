# PV Slurry Formula Performance Prediction
HJT光伏银包铜浆料配方机器学习预测工具

## 项目概述
本项目是面向HJT导电浆料研发的自研数据建模流水线，基于Python实现完整实验数据清洗、工艺特征工程、多目标回归模型训练。针对企业原始配方格式混乱、人工筛选配方耗时久的痛点，自动化完成数据标准化、特征筛选、多模型对比，同步预测体电阻率与银含量两大核心指标，输出各原料组分影响权重，用于新品配方前置筛选，降低人工试错工作量。

企业真实实验配方数据属于涉密资料，本仓库仅开放完整处理逻辑代码，原始Excel数据源未上传。

## 功能模块介绍
### 1. data_clean.py 数据清洗与统计分析
1. 解析零散多行原始实验记录，重构为统一结构化配方表格；
2. 清理特殊空格、统一缺失标记，规整56份实验物料名称与单位；
3. 计算组分与性能相关性，输出热力图、分布直方图、材料对比箱线图；
4. 统计各类粉体、树脂、助剂使用频次，输出材料汇总统计报表。

### 2. feature_engineering.py 特征工程处理
1. 分层填充缺失值：配比空值填充0，物料名称空值标记「未添加」；
2. 结合浆料导电机理构建粉胶比、纳米银占比等8项工艺衍生特征；
3. 物料名称独热编码，将文本物料转为数值特征；
4. 通过方差、相关性双重过滤，剔除无区分度、高度冗余特征。

### 3. model_train.py 多目标机器学习建模
1. 数据集按8:2划分训练集、测试集，完成特征标准化；
2. 并行训练线性回归、随机森林、XGBoost、LightGBM四类回归模型；
3. 采用R²、MAE、RMSE综合评估模型效果，自动筛选最优模型；
4. 持久化保存模型与标准化算子，输出Top关键特征权重可视化图表。
